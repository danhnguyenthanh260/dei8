# dei8

Open-source UI component library — copy-paste style.

Animated SVG components and UI sections built with React, TypeScript, and Tailwind CSS.

## Usage

Browse components at the docs site, copy the source directly into your project. No install required.

## Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Next.js 16 (docs site)

## Registry

Components live in `/registry`. Each folder is self-contained — copy the file(s) you need.

```
registry/
  svgs/
    airplane/       ← multi-angle airplane SVG variants
    package-box/    ← animated shipping box (butterfly-flap)
    jet/            ← jet with cloud effects
    birds/          ← animated birds
    clouds/         ← layered cloud animations
  components/
    sections/       ← full page sections
```

## Contributing

PRs welcome. Keep components self-contained with minimal external dependencies.
